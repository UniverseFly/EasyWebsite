# Easy Website 首页

### 支持 Markdown 格式

- Item A
- Item B
  - Item BA
  - Item BB

### Code Block

```c++
#include <iostream>

int main() {
	std::cout << "Hello World!" << std::endl;
}
```

### 支持数学公式

行内公式 $f(x) = 2x + \frac 1 {e^{-x} + 1}$。

整行公式：
$$
n! = \begin{cases}
1 &n = 0\\
n \cdot (n - 1)! &n \in \mathbb N^+
\end{cases}
$$
